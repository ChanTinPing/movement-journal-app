# 运动日记

一个为手机尺寸设计的本地训练记录应用。应用采用本地存储，适合封装成 PWA 后安装到手机主屏幕自用。

## 已实现

- 首页直接显示历史长卡片
- `+` 新增今天的记录
- 一天内可添加多个动作
- 动作内可按负荷分组，例如 `深蹲 -> 10kg -> 9 / 9 / 9`
- 不限制记录内容，可写 `9`、`60s`、`12`
- 点击历史卡片可继续编辑
- 数据保存在浏览器本地存储
- 支持导出/导入本地 JSON 备份，不上传服务器
- 支持运动日历，有运动记录的日期会标绿

## 本地运行

```bash
npm install
npm run dev
```

## 生产构建

```bash
npm run build
```

## 手机安装

1. 推送到 GitHub 后，GitHub Actions 会把 `dist` 自动部署到 GitHub Pages。
2. Android Chrome：打开地址后使用“安装应用”或“添加到主屏幕”。
3. iPhone Safari：打开地址后点击分享按钮，选择“添加到主屏幕”。

安装后数据仍保存在手机浏览器/主屏幕应用的本地存储中，不会上传到服务器。

当前部署地址：

```text
https://chantinping.github.io/movement-journal-app/
```

## 换手机备份

在旧手机里点“导出”，保存生成的 JSON 文件。把这个文件通过蓝牙、数据线、微信文件传输助手、邮箱草稿等方式移到新手机后，在新手机打开应用点“导入”，选择这个 JSON 文件即可恢复记录。
